## 1. 프로젝트 동기

클라이밍 관련 서비스를 **바이브코딩만으로** 만든다. "안 된다"는 없고, 바이브로 어떻게든 해결하는 것이 원칙이다. AI가 현재 어디까지 할 수 있는지 확인하는 것 자체도 프로젝트의 목적이다.

## 3. 핵심 컨셉 & 가치

> **"유저의 휴먼 리소스를 최소화한다."**

대부분의 과정을 앱이 알아서 수행한다. 사용자는 최소한의 조작만 하면 된다. 유저의 사용자 편의성을 극대화하고, 대부분을 어플리케이션이 자동으로 수행하기를 기대한다.

---

## 4. 사용자 플로우

```
[초기 설정]
   └─ 소셜 로그인 (Apple / Google)

[일상 플로우]
   클라이밍 다녀옴
       ↓
   앱 실행
       ↓
   앱이 자동으로 당일 클라이밍 영상 감지
   (암장 GPS DB + 영상 메타데이터 매칭)
       ↓
   영상을 분석 서버로 업로드
       ↓
   앱 닫고 일상 복귀
       ↓
   ── 분석 서버에서 AI 처리 중 ──
       ↓
   푸시 알림: "분석 완료!"
       ↓
   앱에서 결과 확인
   (클립 목록 + 난이도 + 성공/실패 필터)
       ↓
   올릴 클립 선택
       ↓
   편집된 캐러셀 미리보기
       ↓
   Share Sheet → 인스타그램 앱으로 전달
       ↓
   인스타에서 "게시" 버튼만 누르면 끝
```

## 5. 타겟 유저 & 문제 정의

- **타겟:** 클라이밍 후 인스타그램에 영상을 올리고 싶은 클라이머
- **해결하는 문제:** 촬영한 영상을 일일이 편집하고, 난이도·암장별로 정리하고, 인스타 규격에 맞추는 번거로움
- **핵심 가치:** 클라이밍장에서 찍은 영상만 있으면, 나머지는 앱이 전부 알아서 해준다

## 6. 용어 정의

| **홀드** | 클라이밍 벽에 붙어있는 손잡이 |

## 8. 기능 정의 — PHASE 1 (MVP)

### 8-1. 갤러리 접근 & 영상 수집 (iOS 앱 · iPhone 타겟)

| # | 기능 | 상세 | 비고 |
|---|------|------|------|
| 1-1 | 소셜 로그인 | Apple / Google 로그인 | 프로필 없음, 최소 구현 |
| 1-2 | 영상 자동 필터링 | PHAsset의 `location` + `creationDate`로 당일 클라이밍 영상 감지 | 암장 GPS DB 매칭 |
| 1-3 | 영상 업로드 | 필터링된 영상을 분석 서버로 전송 | 업로드 후 앱 종료 가능 |

**영상 메타데이터 활용 방식:**
- iOS Photos 프레임워크의 `PHAsset` 객체에서 `location`(CLLocation)과 `creationDate`를 추출
- 사전 등록된 암장 GPS 좌표와 비교하여 클라이밍 영상 판별
- 암장 GPS DB는 개발자가 직접 구축 및 관리

### 8-2. 분석 서버 (GPU)

| # | 기능 | 상세 | 비고 |
|---|------|------|------|
| 2-1 | 클립 추출 | raw video → 벽에 붙어 있는 구간만 clip video로 분리 | AI 자동 감지 |
| 2-2 | 성공/실패 분류 | 완등 여부 자동 판별 | 꼭대기 도달+유지 또는 탑아웃=성공, 추락=실패 |
| 2-3 | 난이도 태깅 | 테이프 색상 인식 → 암장별 색상-난이도 매핑 테이블로 자동 태깅 | 매핑 테이블은 개발자가 관리 |
| 2-4 | 사용자 식별 | 그날 영상에서 가장 많이 등장하는 등반자 = "나" | 실시간 적응형 학습 |
| 2-5 | 영상 편집 | 캐러셀용 편집 (세로 3:4, 클립당 최대 60초) | 인스타 규격 준수 |

**성공/실패 판별 기준:**
- 성공: 최상단 홀드 도달 후 일정 시간 유지, 또는 탑아웃
- 실패: 등반 중 추락 (스스로 내려오면 성공)

**사용자 식별 방식:**
- 삼각대 고정 촬영이므로 촬영자는 프레임에 미등장
- 영상에 등장하는 여러 등반자 중 출현 빈도가 가장 높은 사람을 "나"로 판별

### 8-3. 결과 표출 & 공유

| # | 기능 | 상세 | 비고 |
|---|------|------|------|
| 3-1 | 분석 완료 알림 | 푸시 알림으로 결과 도착 안내 | 업로드 후 앱 종료 가능 |
| 3-2 | 결과 목록 | 모든 클립 + 메타데이터(난이도, 성공/실패) 표출 | 필터 기능 포함 |
| 3-3 | 클립 미리보기 | 선택한 클립 재생 | |
| 3-4 | 캐러셀 구성 | 올릴 클립 선택 + 순서 정하기 | 여러 클립을 한 게시물로 |
| 3-5 | 인스타 공유 | Share Sheet로 인스타그램 앱에 캐러셀 전달 | 유저는 "게시"만 누르면 끝 |
